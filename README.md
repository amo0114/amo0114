## 🧭 关于我

我更在意把项目做成 **“线上可跑、可观测、可迭代”** 的工程化作品，而不仅是功能堆砌。

- 💻 方向：后端开发 / 系统设计 / DevOps 工程化
- 🧠 关注：缓存与性能、消息队列、可靠性、CI/CD 与容器化
- 🚀 目标：从开发者 → 系统架构设计能力持续进化
