## 🧭 关于我

我把系统当作一座会呼吸的城市：  
白天承载流量，夜里修补裂缝；  
上线不是句号，而是一段漫长叙事的开场。

- 💻 方向：后端开发 / 系统设计 / DevOps 工程化  
- 🧠 关注：缓存与性能、消息队列解耦、可靠性、CI/CD 与容器化交付  
- 🚀 目标：把“写得出来”变成“跑得稳、看得清、迭得快”

---

## 🚀 代表项目（建议放真实链接）

- 🏠 **房屋租赁系统 / Rent House System**：Spring Boot + Vue3 全栈（缓存、MQ、Docker、工程化）
- 🍲 **Foodie Platform**：位置服务 + 高并发列表/分页策略 + 缓存协同
- 🤖 **MoGPT**：类 ChatGPT 的 AI 交互应用（会话模型 / 上下文 / 流式体验）

> 如果你愿意，把 3 个项目仓库链接发我，我可以把每条项目改成“更像作品集”的一句话：**一句话定位 + 两个亮点 + 技术栈标签**。
